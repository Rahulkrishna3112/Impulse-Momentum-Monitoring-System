# 💳 Impulse Momentum Monitoring System (IMMS)

## 🚀 Overview

The Impulse Momentum Monitoring System is a Behavioural Analytics Engine designed to detect impulse build-up phases in financial behaviour.

Instead of identifying overspending after it occurs, this system predicts behavioural momentum escalation before high-risk spending bursts.

---

## 🧠 Core Innovation

We introduce an **Impulse Momentum Score (0–100)** based on:

- Spending Acceleration
- Behavioural Volatility
- Transaction Time Compression
- Night Spending Ratio
- Category Drift

Users transition through behavioural phases:

Stable → Vulnerable → Build-Up → High-Risk → Burst

The system also detects phase escalations and generates intelligent behavioural interventions.

---

## 🏗 System Architecture

1. User Behaviour Simulation Engine
2. Semi-Real-Time Transaction Generator
3. Daily Behaviour Metrics
4. Rolling Behaviour Intelligence
5. Impulse Momentum Scoring
6. Phase State Machine
7. Intervention Engine
8. Interactive Risk Dashboard

---

## 📊 Dashboard Features
